# GymChuot Roadmap

> "Garmin Connect cho PT Việt Nam — nhưng giao diện đẹp hơn và có chat."

## Phase 1 — Foundation (Now → 1 month)
- [x] Workout Builder UI (drag & drop, per-set config)
- [x] Messenger UI (chat + workout card)
- [x] Exercise data pipeline (800+ bài từ free-exercise-db)
- [ ] Auth — PT vs Student roles
- [ ] Backend + DB (Supabase)
- [ ] Real-time messaging

## Phase 2 — Core Product (Month 2-3)
- [ ] Student Workout Viewer (rest timer, log actual results)
- [ ] Workout Templates (save + clone)
- [ ] Video attachment (≤ 25MB)
- [ ] Push notifications
- [ ] Advanced exercise search (muscle group, equipment, level)

## Phase 3 — Engagement (Month 4-6)
- [ ] Progress tracking + charts
- [ ] PT Dashboard (multi-student)
- [ ] Workout history with date filter
- [ ] Exercise notes per student
- [ ] Mobile PWA (installable Android/iOS)

## Phase 4 — Community & Scale (Month 6+)
- [ ] Community exercise contributions (PT submit video + commonMistakes)
- [ ] Exercise verification system
- [ ] PT Marketplace (public profile, student discovery)
- [ ] Workout Programs (multi-week plans, sellable)
- [ ] AI workout suggestions
